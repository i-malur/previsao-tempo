
# 🌤️ Previsão do Tempo

Aplicação web desenvolvida para consultar a previsão do tempo em diferentes cidades utilizando uma API externa. O projeto consome dados em tempo real e exibe informações climáticas de forma dinâmica e visual.

Projeto criado através do vídeo:  
[Criando um App de Previsão do Tempo com HTML, CSS e JavaScript](https://www.youtube.com/watch?v=qxzqEuAOYZ4)

## 🚀 Funcionalidades

- 🔎 Busca de cidade (por voz e texto)
- 🌡️ Exibição de temperatura atual
- ☁️ Exibição das condições climáticas
- 🏙️ Exibição do nome da cidade pesquisada
- 🖼️ Background dinâmico relacionado ao clima/local
- 📱 Layout responsivo
- 🤖 Sugestões de roupas com a IA Inês

## 🔌 Como funciona

A aplicação realiza uma requisição para uma API de previsão do tempo utilizando `fetch()`, recebe os dados em formato JSON e atualiza dinamicamente os elementos do DOM com as informações retornadas.

## 📦 Como executar o projeto

1. Clone o repositório: `git clone https://github.com/seu-usuario/previsao-tempo.git`
2. Abra o arquivo `index.html` no navegador  
ou utilize a extensão Live Server no VS Code.
